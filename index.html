Scalable Vector Graphics - формат векторных изображений, основанный на XML.

Преимущества формата

  • Масштабируется
  • Четкий на ретине
  • Человекопонятный код
  • Можно написать руками
  • Исходник
  • Переиспользуемые компоненты
  • Внутри - CSS и JavaScript

Способы вставки

  • Font

Какой лучше?

  • инлайновый SVG - для оформления внешними стилями и управления через JS
  • SVG-спрайт - если нет необходимости управления через внешний CSS
  • иконочный шрифт - удобно подключать и использовать, но проблемы с поддержкой браузерами
