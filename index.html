Что такое SVG?

Scalable Vector Graphics - формат векторных изображений, основанный на XML.

Преимущества формата

  • изображения масштабируются без потери качества
  • не теряют резкости на ретине
  • файл можно открыть в текстовом редакторе и увидеть человекопонятный код
  • SVG можно не только нарисовать, но и написать
  • SVG-файл - это исходник
  • можно создавать реиспользуемые компоненты
  • внутри файла могут находиться CSS и JavaScript

Какой лучше?

  • инлайновый SVG - для оформления внешними стилями и управления через JS
  • SVG-спрайт - если нет необходимости управления через внешний CSS
  • иконочный шрифт - удобно подключать и использовать, но проблемы с поддержкой браузерами
