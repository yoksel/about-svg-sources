Что такое SVG?

Scalable Vector Graphics - формат векторных изображений, основанный на XML.

Преимущества формата

  • Масштабируется
  • Четкий на ретине
  • Человекопонятный код
  • Можно написать руками
  • Исходник
  • Переиспользуемые компоненты
  • Внутри - CSS и JavaScript
